HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • A standardized system for tagging text files to achieve font, color, graphic, and hyperlink effects on World Wide Web pages

CSS

  • Contains various design languages for webpages.
  • Gives a webpage it's "vibe".
  • Can be used for formatting page.

Git

  • Imperative to the iterative control process.
  • Allows multiple users to code without conflict.
  • git status: Check what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • Funny name.

JavaScript

  • Creates interactive features on webpage.
  • Allows users to control functionality of webpages.
  • A variable is a named container that allows us to store data in our code.
  • Control flow in the order in which a computer executes code in a script.
